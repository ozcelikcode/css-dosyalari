calc() : Değer hesaplama işlemleri için kullanılır.
Değerler : toplama (+), çıkarma (-), çarpma (*), bölme (/)

Width değeri için calc() kullanımı

calc(50% + 100px) = genişliği 50% ver. 50% ile 100px'i topla.

calc(50% - 100px) = genişliği 50% ver. 50%'den 100px çıkar.

calc(50% * 2) = genişliği 50% ver. 50%'yi 2 ile çarp. 2px ile değil -> 2 ile olacaktır unutmayın!

calc(50% / 2) = genişliği 50% ver. 50%'yi 2 ile böl.

Height değeri için calc() kullanımı

bilgi : height-box kutusu height değeri 350px

calc(50% + 100px) : yüksekliği 50% ver. 50% ile 100px'i topla.
calc(50% - 100px) : yüksekliği 50% ver. 50%'den 100px çıkar.
calc(50% * 100px) : yüksekliği 50% ver. 50%'yi 2 ile çarp.
calc(50% - 100px) : yüksekliği 50% ver. 50%'yi 2 ile böl

Örnek kullanım

genişlik 100px
genişlik 100% ver. 100%'den 100px çıkar.
