Yazıları göremediğin nesnelerin üzerine gel. Açıklama mevcut.

transform: rotate(değer)

rotate sayesinde nesneleri döndürebilirsin. Mesela transform: rotate(90deg); yazarsan nesne '90 derece' döner.

transform: skew(değer)

nesneyi sağ-sol taraftan çevirir. Mesela transform: skew(-30deg);

transform: scale(değer)

nesneyi yazdığın kat kadar büyütür. Mesela transform: scale(2) > nesneyi 2 kat büyütür. scale(1) varsayılan değerdir.

transform: translate(x, y, z)

translate sayesinde nesneyi olduğu yerden hareket ettirebilirsin.
